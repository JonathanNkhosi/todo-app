# Todo App (Java Console)

A simple Java console-based To-Do List application that allows users to manage tasks efficiently.

## Features

- Add tasks
- View all tasks
- Delete tasks
- Menu-driven interface
- Input validation

## Technologies Used

- Java
- ArrayList
- Scanner (User Input)
- Switch statements
- Loops & Functions

## How to Run

1. Compile the program:

javac TodoApp.java

2. Run the program:

java TodoApp

- How to design menu-driven programs
- Using functions to organize code
- Working with ArrayList
- Input handling and validation
